# TrustMRR

TrustMRR Raycast extension. Browse startups directly from Raycast.

## Commands

- `List Startups`: List all startups with pagination and filtering
- `Get Startup`: Get full startup details by slug.

## Setup

1. Generate an API key in TrustMRR. [https://trustmrr.com/dashboard-dev](https://trustmrr.com/dashboard-dev)
2. When you use any of the commands, you'll be prompted to enter your API key. Or you can set it in the extension preferences for TrustMRR extension.
